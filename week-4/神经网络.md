#### 感知器学习规则
如下的一个模型

![image](https://github.com/jccjd/Coursera-Machine-Learning/blob/master/week-4/image/神经网络3.PNG?raw=true)

假设：
    t = 1, n = 1, x1 = 1, w1 = -5, b = 0
    
Step1:
    y = sign(1 * (-5)) = -1
    △w = 1 * (1 - (-1))* 1 = 2
    w1 = w1 + △w = -3
     
Step2:
    y = sign(1 * (-3)) = -1
    △w = 1 * (1 - (-1))* 1 = 2
    w1 = w1 + △w = -1
    
Step3:
    y = sign(1 * (-1)) = -1
    △w = 1 * (1 - (-1))* 1 = 2
    w1 = w1 + △w = 1
    y = sign(1*1 ) = 1 = t
然后迭代结束 预测值 = 真实值
